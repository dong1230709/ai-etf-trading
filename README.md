# AI ETF 交易系统

专业级AI驱动ETF网格交易终端

## 功能特性

- 📊 **实时行情** - 多数据源实时行情监控
- 🎯 **网格交易** - 专业级网格交易系统
- 🤖 **AI分析** - 智能市场分析和交易建议
- ⚠️ **风险控制** - 全方位风险监控
- 📱 **移动优先** - 响应式设计，完美支持移动端
- ⚡ **PWA支持** - 可安装至手机桌面

## 技术栈

- React 18 + TypeScript
- Vite
- TailwindCSS
- Zustand (状态管理)
- Framer Motion (动画)
- React Router DOM

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 类型检查
npm run check
```

## 页面导航

- `/` - 首页 Dashboard
- `/grid` - 网格交易
- `/grid-engine` - 专业网格引擎
- `/ai-plan` - AI交易计划
- `/risk` - 风险中心
- `/settings` - 设置
